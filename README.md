# `hr-breaker` local setup

The repo includes a reusable local wrapper around `hr-breaker` for ATS-oriented PDF optimization.

1. Install and pin the tool locally:
   `scripts/setup-hr-breaker.sh`
2. Add an API key to repo-local `.env` or directly to `tools/hr-breaker/.env`
3. Run it against an existing resume PDF and a vacancy file:
   `scripts/run-hr-breaker.sh "dist/Sergei Afonchenko - Backend Developer.pdf" "vacancies/ggr-global-senior-go-developer.md" ru`

Notes:
- The setup pins `hr-breaker` to a known commit for reproducibility.
- `scripts/run-hr-breaker.sh` will source the repo-local `.env` automatically if it exists.
- Input should be the existing PDF resume from `dist/`.
- Output is a new optimized PDF generated by `hr-breaker` under its own `output/` directory.
