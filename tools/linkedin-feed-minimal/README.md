# LinkedIn Feed Minimal

This folder contains a Stylus userstyle for a stripped-down LinkedIn feed.

Setup:

1. Install the `Stylus` browser extension.
2. Open [linkedin-feed-minimal.styl](/home/user/repos/sergeia-resume-full-stack/tools/linkedin-feed-minimal/linkedin-feed-minimal.styl).
3. Copy the file contents into a new Stylus style, or import the file directly.
4. Open `https://www.linkedin.com/feed/` and refresh the page.

What it does:

- collapses feed posts to the engagement row
- hides many promoted cards
- hides the right-rail block with `Today’s puzzles` and `Add to your feed`
- hides `#ads-container`

The selectors are based on LinkedIn’s current DOM and may need updates later.
